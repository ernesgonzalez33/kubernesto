Projects

Here are some of the projects I have worked on:

  • Pod Escape: Endless Runner Game developed using Godot in which you are a Kubernetes pod trying to escape from the deletion of the namespace you were deployed. Co-designer and Co-developer. Playable here.
  • Marinera: A subset of the FIWARE platform with CI/CD tooling to make it easier to deploy on top of OpenShift. Contributor.
  • Writing: An app that writes continously to /tmp created for demoing Resource Quotas on Kubernetes. Developer.
  • OpenShift Unused Projects: Set of scripts created to get an overview of some OpenShift projects are probably not being used. Developer.
  • Overworld Tactics: Tactics game with two levels and three difficulties created for my final project at University. Developed using LibGDX. Designer and Developer. Downloadable here
